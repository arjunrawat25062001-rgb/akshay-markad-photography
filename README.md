# Akshay Markad Photography

Production-ready monorepo scaffold for the Akshay Markad Photography website.

## Structure

- `frontend` — Next.js 15, TypeScript, Tailwind CSS, App Router, and ESLint.
- `backend` — Spring Boot 3 API targeting Java 21.
- `database` — migrations and database resources.
- `docs` — project documentation.
- `.github` — GitHub configuration and workflows.

## Prerequisites

- Node.js 20.9 or later
- Java 21
- Maven 3.9 or later

## Commands

```sh
npm install
npm run dev
cd backend && mvn spring-boot:run
```
